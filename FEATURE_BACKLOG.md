# GF In The Field - Feature Backlog

## 🚀 Planned Enhancements

### Voice-to-Text Observation Entry
**Priority:** Nice to have  
**Added:** 2026-05-27  
**Requested by:** Ben

**Description:**
Add speech-to-text capability for observation notes. FSMs can speak their observations instead of typing—app auto-transcribes and fills text fields.

**Benefits:**
- Faster observation capture in the field
- More natural for FSMs doing observations on mobile
- Reduces friction—speaking is easier than typing on phone screens

**Technical Considerations:**
- Mobile browser Web Speech API (free, works in Chrome/Safari)
- Fallback option: Continue allowing typed input
- Privacy: Speech processing happens on-device (no audio sent to servers)

**Implementation Notes:**
- Add microphone button to each observation text field
- Show real-time transcription as FSM speaks
- Let FSM edit transcribed text before saving
